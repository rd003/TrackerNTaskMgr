# TrackerNTaskMgr

## Tech stack

### Backend

- C# 
- Dotnet 9 Web API 
- Sql Server 2022 (database)
- Dapper (ORM)
- Rd003.JwtAuth (A [untility library](https://www.nuget.org/packages/Rd003.JwtAuth/1.0.0) for creating JWT tokens)
- BCrypt.Net-next for password hashing

### Front End

- Angular 19
- Angular material 19
